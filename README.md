# 💎 Free Fire Bind E-mail Script

A complete utility script for managing Free Fire account bindings using access tokens. This tool provides multiple features related to email binding, account info, and token management.

---

## ✨ Features

- ✅ Check Bind Information  
- 📧 Bind Email  
- ❌ Unbind Email  
- 🔄 Change Bind Email  
- 🚫 Cancel Bind Request  
- 🔑 Get Access Token  
- ♻️ Revoke Access Token  
- 📜 Get Login History  
- 🔗 Check Bound Accounts  

---

## 🚀 Usage

1. Clone or download this repository.
2. Install required dependencies:
   ```bash
   pip install -r requirements.txt

3. Run the script:

python main.py


4. Enter your Access Token when prompted.


5. Select the desired option from the menu.
